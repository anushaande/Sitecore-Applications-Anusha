What is this?
Add New Button(*)
[[Text|Link::New|Button_Style|Show_Right_Arrow]]

Location Expected Value
Text Button Text (eg. Click Here)
Link Target location (eg. http://health.usf.edu)
::New (optional) Open target location in a new tab (eg. http://health.usf.edu::New)
Button_Style Available Styles:
  • bg-usfGreen
  • bg-usfApple
  • bg-usfSky
  • bg-usfStorm
  • bg-usfTealGreen
  • bg-usfSlate
  • bg-usfSeaGlass
Show_Right_Arrow (optional) Enable "Right Arrow" symbol (True or False)
Default is True
Sample Code - [[Text|Link::New|Button_Style|Show_Right_Arrow]] Output
[[USF Green|http://yourTargetURL::New|bg-usfGreen|True]] USF Green
[[USF Apple|http://yourTargetURL::New|bg-usfApple|True]] USF Apple
[[Sky Blue|http://yourTargetURL|bg-usfSky|False]] Sky Blue
[[Storm|http://yourTargetURL::New|bg-usfStorm]] Storm
[[Teal Green|http://yourTargetURL|bg-usfTealGreen|True]] Teal Green
[[Slate|http://yourTargetURL|bg-usfSlate|False]] Slate
[[Seaglass|http://yourTargetURL|bg-usfSeaGlass|True]] Seaglass
